# Enterprise Network Redesign & Modernization

## 📌 Objective
Modernized a legacy "flat" network architecture for an SMB to improve security, scalability, and performance. The redesign focuses on eliminating single points of failure and implementing strict access control between functional zones.

## 🛠️ Modernization Key Features
* **From Flat to Segmented:** Transitioned from a single-subnet LAN to a multi-VLAN structure (Sales, HR, IT, Guest) to reduce the blast radius of potential attacks.
* **Redundancy & High Availability:** Integrated dual-homed ISP connections and failover firewalls to ensure 99.9% uptime.
* **Legacy Hardware Decommissioning:** Replaced aging infrastructure with modern Next-Generation Firewalls (NGFW) capable of Deep Packet Inspection (DPI).
* **Remote Workforce Support:** Implemented Split-Tunnel VPNs and Multi-Factor Authentication (MFA) for secure remote access.
